闭包：简言之，就是函数里面的函数

闭包的好处：
使得函数内的变量一直保存在内存里面，避免被javascript回收机制回收
从而保护变量的安全性

闭包的形成原理：是基于函数变量作用域的规则 和 垃圾回收机制规则。 

例子：
function a() {
    var i=0;
    function b() {
        console.log(++i);
    }

    return b;
}
var c=a();
c();
c();
c();

输出结果是： 1 2 3

若没有用闭包的话，每次重新执行同一个函数，返回的结果都是一样的（每次执行完被回收）

避免滥用闭包，容易造成内存不断变大。
